# HTML CSS JavaScript Demo Lessons

This ZIP contains ready-to-teach demo code for beginner web programming lessons.

## How to use

1. Unzip the file.
2. Open `index.html` in Google Chrome or any browser.
3. Click each lesson demo.
4. For JavaScript lessons, open browser DevTools Console to see extra output.

## Folder structure

```text
web_demo_code_lessons/
├── index.html
├── assets/
│   ├── css/main.css
│   ├── js/main.js
│   └── images/logo.svg
├── lessons/
│   ├── html/
│   ├── css/
│   └── javascript/
└── projects/
```

## Teaching order

1. HTML structure and common tags
2. CSS styling and layout
3. JavaScript basics
4. DOM and events
5. Mini projects

## Tip for teacher

Ask students to edit the code after each demo:

- Change text and colors
- Add one new section
- Add one button interaction
- Make the page responsive
- Build a small final project
