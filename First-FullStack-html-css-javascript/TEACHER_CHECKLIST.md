# Teacher Checklist

## HTML practice tasks

- Create a personal introduction page.
- Add heading, paragraph, link, image, list, table, and form.
- Use semantic tags: header, nav, main, section, footer.

## CSS practice tasks

- Change background and text color.
- Create a card with padding, border, margin, and shadow.
- Build a navbar using Flexbox.
- Build a gallery using CSS Grid.
- Make the page responsive for mobile.

## JavaScript practice tasks

- Use variables and show data in HTML.
- Use if/else to check pass/fail.
- Use loop to print numbers or list data.
- Use array to manage students.
- Use object to store one student profile.
- Use DOM to change text, color, and class.
- Use event listener for button click and input typing.
- Validate a form.
- Build Todo, Tabs, Modal, Slider, and Quiz.

## Final project ideas

- Student portfolio website
- School course landing page
- Product page
- Quiz app
- Todo app
- Simple admin dashboard UI
