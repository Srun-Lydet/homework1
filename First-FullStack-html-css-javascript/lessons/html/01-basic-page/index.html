HTML 01 - Basic Page Structure

Basic HTML Document

Every webpage starts with a document structure: doctype, html, head, and body.

Hello Students!

This is my first HTML page.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Page</title>
</head>
<body>
  <h1>Hello Students!</h1>
</body>
</html>
